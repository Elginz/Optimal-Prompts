# React / Next.js Project Scaffolding Prompt

This prompt is used to generate the **initial structure, configuration, and boilerplate** for a React / Next.js web application.  
It is intended for **project setup and planning**, not full feature implementation.

Follow the instructions below when using the prompt.

---

## Purpose

Use this prompt to:
- Define platform and environment requirements
- Design application pages and routing
- Generate a recommended Next.js project file structure
- Create appropriate `.gitignore` files
- Produce minimal boilerplate code to start development

---

## How to Use the Prompt

Replace all placeholder values enclosed in `<< >>` with concrete project decisions.

Do **not** leave placeholders empty.

---

## 1. Application & Platform Specifications

Describe the environment and constraints of your application.

### 1.1 Operating System Target
Specify the operating system(s) the application is intended to run on.

Examples:
- Cross-platform (Windows, macOS, Linux) via modern browsers
- Android 10+ via PWA or WebView
- Desktop-only browsers
